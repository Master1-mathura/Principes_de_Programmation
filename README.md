# Principes de Programmation

Ce répertoire rassemble l'ensemble de mes notes de cours et travaux pratiques réalisés dans le cadre du module **Principes de Programmation**. 

Il représente la **base théorique et technique** qui a permis la conception et la réalisation du projet Don't WatchList.

**[Cliquez ici pour découvrir le projet DWL](https://github.com/Master1-mathura/Projet_DWL.git)**

## Arborescence du Dépôt

Le dépôt est structuré de manière très simple pour aller à l'essentiel :
```text
/
├── PP_TP/              # Dossier principal
│   ├── 1 dossier par notion abordée
│   └── README.md       # Mes notes de cours
└── .gitignore          # Fichiers ignorés par Git
```
